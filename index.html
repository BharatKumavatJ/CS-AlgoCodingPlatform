Syllabus for Interview Prepration Course
Course Curriculum
1. Fundamentals Of Programming
About
This module is designed to introduce you to programming. You'll learn how to communicate with a computer.
Topics Covered
Basics, Conditionals, Loops, Functions, Arrays, 2D Arrays, Time & Space Complexity.
2. Getting Started with Data Structure
About
This module will help you get started with learning about data structures and algorithms.
Topics Covered
Recursion, Sorting Algorithms : Merge Sort and Quick Sort, Binary Search, Binary Search on Answers.

Time Complexity and space complexity analysis of each algorithm and Interview questions.
3. Intermediate Data Structure
About
This module will focus on intermediate data structures and various problem solving techniques.
Topics Covered
Linked List, Two pointer, Sliding Window, Kadane's Algorithm, Stack, Queue, Deque, Hashing and Heaps.
Applications and Interview Questions.
4. Computer Science Fundamentals
About
In this module we will cover computer science fundamentals.
Topics Covered
Object Oriented Programming, Operating System, Database management, Sql, Networking.

Most ask Interview Questions.
5. Advance Data Structure
About
This module will focus on advance data structures and algorithm.
Topics Covered
BIt Manipulation, Tree, Graph, Greedy Algorithms and Dynamic Programming.

Applications and Interview Questions.
